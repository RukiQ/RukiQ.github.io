一、整体预览

二、引用类型

引用类型是一种数据结构，用于将数据和功能组织在一起。

对于 ECMAScript 中的引用类型而言，prototype 是保存它们所有实例方法的真正所在。

1、Object 类型

创建 Object 实例有两种方式，一种是使用 new 操作符后跟 Object 构造函数，另一种方式是使用对象字面量表示法。通过对象字面量定义对象时，实际上不会调用 Object 构造函数。

访问对象属性时，有两种方法：点表示法 和 方括号表示法。通常，除非必须使用变量来访问属性，否则建议使用点表示法。

2、Array 类型

数组的 length 属性很有特点——它不是只读的。因此，通过设置这个属性，可以从数组的末尾移除项或向数组中添加新项。

（1）检测数组

Array.isArray(value)

忽略变量创建的全局作用域

（2）转换方法

所有对象都具有 toLocaleString()、toString()、valueOf() 方法。调用 valueOf() 返回的还是数组。

join
str = arr.join([separator = ','])

（3）栈方法

push
arr.push(element1, ..., elementN)

push() 方法可以接受任意数量的参数，把它们逐个添加到数组末尾，并返回修改后数组的长度。

pop
array.pop()

pop() 方法删除一个数组中的最后的一个元素，并且返回这个元素。

（4）队列方法

shift
arr.shift()

shift() 方法删除数组的 第一个 元素，并返回这个元素。该方法会改变数组的长度。

unshift
arr.unshift()

在数组前端添加任意项兵返回新数组的长度。

（5）重排序方法

reverse
arr.reverse()

反转数组

sort
arr.sort([compareFunction])

sort() 方法会调用每个数组项的 toString() 转型方法，然后比较得到的字符串。默认升序排列。

（6）操作方法

concat
array.concat(value1, value2, ..., valueN)

将传入的数组或非数组值与原数组合并,组成一个新的数组并返回.

slice
array.slice(begin[, end])

slice() 方法把数组中一部分的浅复制存入一个新的数组对象中，并返回这个新的数组。
如果结束位置小于起始位置，则返回空数组。

splice
array.splice(start, deleteCount[, item1[, item2[, ...]]])

splice() 方法用新元素替换旧元素，以此修改数组的内容。主要用途是向数组的中部插入项。返回由被删除的元素组成的数组（包括一个元素的数组和空数组）。

（7）位置方法

indexOf
arr.indexOf(searchElement[, fromIndex = 0])

indexOf() 方法返回给定元素能找在数组中找到的第一个索引值，否则返回-1。

lastIndexOf
arr.lastIndexOf(searchElement[, fromIndex = arr.length - 1])

lastIndexOf() 方法返回指定元素（也即有效的 JavaScript 值或变量）在数组中的最后一个的索引，如果不存在则返回 -1。从数组的后面向前查找，从 fromIndex 处开始。

（8）迭代方法

  1. ES5 为数组定义了5个迭代方法。每个方法都接收两个参数：要在每一项上运行的函数和（可选的）运行该函数的作用域对象——影响 this 的值。
  2. 传入这些方法中的函数（callback）会接收三个参数：数组项的值、该项在数组中的位置和数组对象本身。
    —>callback(item, index, array)
  3. callback 执行后的返回值可能会也可能不会影响方法的返回值。
  4. 迭代方法都不会修改数组中的包含的值。
every
arr.every(callback[, thisArg])

every() 方法测试数组的所有元素是否都通过了指定函数的测试。如果该函数对每一项都返回 true，则返回 true。

filter
arr.filter(callback[, thisArg])

filter() 方法使用指定的函数测试所有元素，并创建一个包含所有通过测试的元素的新数组。即返回 true 的项组成的数组。

forEach
array.forEach(callback[, thisArg])

forEach() 方法让数组的每一项都执行一次给定的函数。本质上与使用 for 循环迭代数组一样。无返回值。

map
array.map(callback[, thisArg])

map() 方法对数组中的每一项运行给定函数，返回每次函数调用的结果组成的数组。

some
arr.some(callback[, thisArg])

some() 方法测试数组中的某些元素是否通过了指定函数的测试，如果该函数中对任一项返回 true，则返回 true。

（9）归并方法

  1. 归并方法都接收两个参数：一个在每一项上调用的函数和（可选的）作为归并基础的初始值。
  2. callback 接收4个参数，前一个值，当前值，项的索引和数组对象。
    —>callback(prev, cur, index, array)
  3. 这个函数返回的任何值都会作为第一个参数自动传给下一项。
reduce
arr.reduce(callback,[initialValue])

initialValue—>可作为第一次调用回调 callback 的第一个参数

reduce() 方法接收一个函数作为累加器（accumulator），数组中的每个值（从左到右）开始合并，最终为一个值。

一、V8 的垃圾回收机制与内存限制

  1. V8 的内存限制

    在一般的后端开发语言中，在基本的内存使用上没有什么限制，但在 Node 中通过 JavaScript 使用内存时，只能使用部分内存，这样，再单个 Node 进程的情况下，计算机的内存资源无法得到充足的使用。

    ☞ 主要原因：Node 基于 V8 构建，在 Node 中使用的 JavaScript 对象基本上都是通过 V8 自己的方式来进行分配和管理的。
