《JavaScript高级程序设计》第5章 引用类型

发表于 2018-07-28 | 分类于 《JavaScript高级程序设计》

整体预览

引用类型

引用类型是一种数据结构，用于将数据和功能组织在一起。

对于 ECMAScript 中的引用类型而言，prototype 是保存它们所有实例方法的真正所在。

Object 类型

创建 Object 实例有两种方式，一种是使用 new 操作符后跟 Object 构造函数，另一种方式是使用对象字面量表示法。通过对象字面量定义对象时，实际上不会调用 Object 构造函数。

访问对象属性时，有两种方法：点表示法 和 方括号表示法。通常，除非必须使用变量来访问属性，否则建议使用点表示法。

Array 类型

数组的 length 属性很有特点——它不是只读的。因此，通过设置这个属性，可以从数组的末尾移除项或向数组中添加新项。

检测数组

Array.isArray(value)

忽略变量创建的全局作用域

转换方法

所有对象都具有 toLocaleString()、toString()、valueOf() 方法。调用 valueOf() 返回的还是数组。

join
str = arr.join([separator = ','])

栈方法

push
arr.push(element1, ..., elementN)

push() 方法可以接受任意数量的参数，把它们逐个添加到数组末尾，并返回修改后数组的长度。

pop
array.pop()

pop() 方法删除一个数组中的最后的一个元素，并且返回这个元素。

队列方法

shift
arr.shift()

shift() 方法删除数组的 第一个 元素，并返回这个元素。该方法会改变数组的长度。

unshift
arr.unshift()

在数组前端添加任意项兵返回新数组的长度。

重排序方法

reverse
arr.reverse()

反转数组

sort
arr.sort([compareFunction])

sort() 方法会调用每个数组项的 toString() 转型方法，然后比较得到的字符串。默认升序排列。

操作方法

concat
array.concat(value1, value2, ..., valueN)

将传入的数组或非数组值与原数组合并,组成一个新的数组并返回.

slice
array.slice(begin[, end])

slice() 方法把数组中一部分的浅复制存入一个新的数组对象中，并返回这个新的数组。
如果结束位置小于起始位置，则返回空数组。

splice
array.splice(start, deleteCount[, item1[, item2[, ...]]])

splice() 方法用新元素替换旧元素，以此修改数组的内容。主要用途是向数组的中部插入项。返回由被删除的元素组成的数组（包括一个元素的数组和空数组）。

位置方法

indexOf
arr.indexOf(searchElement[, fromIndex = 0])

indexOf() 方法返回给定元素能找在数组中找到的第一个索引值，否则返回-1。

lastIndexOf
arr.lastIndexOf(searchElement[, fromIndex = arr.length - 1])

lastIndexOf() 方法返回指定元素（也即有效的 JavaScript 值或变量）在数组中的最后一个的索引，如果不存在则返回 -1。从数组的后面向前查找，从 fromIndex 处开始。

迭代方法

  1. ES5 为数组定义了5个迭代方法。每个方法都接收两个参数：要在每一项上运行的函数和（可选的）运行该函数的作用域对象——影响 this 的值。
  2. 传入这些方法中的函数（callback）会接收三个参数：数组项的值、该项在数组中的位置和数组对象本身。
    —>callback(item, index, array)
  3. callback 执行后的返回值可能会也可能不会影响方法的返回值。
  4. 迭代方法都不会修改数组中的包含的值。
every
arr.every(callback[, thisArg])

every() 方法测试数组的所有元素是否都通过了指定函数的测试。如果该函数对每一项都返回 true，则返回 true。

filter
arr.filter(callback[, thisArg])

filter() 方法使用指定的函数测试所有元素，并创建一个包含所有通过测试的元素的新数组。即返回 true 的项组成的数组。

forEach
array.forEach(callback[, thisArg])

forEach() 方法让数组的每一项都执行一次给定的函数。本质上与使用 for 循环迭代数组一样。无返回值。

map
array.map(callback[, thisArg])

map() 方法对数组中的每一项运行给定函数，返回每次函数调用的结果组成的数组。

some
arr.some(callback[, thisArg])

some() 方法测试数组中的某些元素是否通过了指定函数的测试，如果该函数中对任一项返回 true，则返回 true。

归并方法

  1. 归并方法都接收两个参数：一个在每一项上调用的函数和（可选的）作为归并基础的初始值。
  2. callback 接收4个参数，前一个值，当前值，项的索引和数组对象。
    —> callback(prev, cur, index, array)
  3. 这个函数返回的任何值都会作为第一个参数自动传给下一项。
reduce
arr.reduce(callback,[initialValue])

initialValue—>可作为第一次调用回调 callback 的第一个参数

reduce() 方法接收一个函数作为累加器（accumulator），数组中的每个值（从左到右）开始合并，最终为一个值。

Date 类型

RegExp 类型

Function 类型

函数实际上是对象。每个函数都是 Function 的实例，而且都与其他引用类型一样具有属性和方法。
“函数是对象，函数名是指针。”

使用不带圆括号的函数名是访问函数指针，而非调用函数。

函数内部属性

在函数内部，有两个特殊对象：arguments 和 this。

arguments

类数组对象，包含着传入函数中的所有参数。
主要用途：保存函数参数。

callee：指针，指向拥有这个 arguments 对象的函数。可以消除函数名的紧耦合。

caller：严格模式下访问 arguments.caller 属性会导致错误，非严格模式下这个属性始终是 undefined。

this

引用的是函数据以执行的环境对象。

caller

caller 属性保存着调用当前函数的函数的引用，如果是在全局作用域中调用当前函数，它的值为 null。为了实现更松散的耦合，也可以通过 arguments.callee.caller 来访问相同的信息。

函数方法

apply
fun.apply(thisArg[, argsArray])

apply() 方法在指定 this 值和参数（参数以数组或类数组对象的形式存在）的情况下调用某个函数。

call
fun.call(thisArg[, arg1[, arg2[, ...]]])

call() 方法在使用一个指定的this值和若干个指定的参数值的前提下调用某个函数或方法。

bind
fun.bind(thisArg[, arg1[, arg2[, ...]]])

bind()方法会创建一个函数的实例，其 this 值会被绑定到传给 bind() 函数的值。

call() 和 apply() 方法的作用相同，它们的区别仅在于接收参数的方法不同。并且，它们真正强大的地方在于能够扩充函数赖以运行的作用域。

基本包装类型

引用类型与基本包装类型的主要区别就是对象的生存期。使用 new 操作符创建的引用类型的实例，在执行流离开当前作用域之前一直保存在内存中。而自动创建的基本包装类型的对象，则只存在于一行代码的执行瞬间，然后立即被销毁。

Number 方法

toFixed
numObj.toFixed(digits)

toFixed() 会按照指定的小数位返回数值的字符串表示。

toExponential
numObj.toExponential(fractionDigits)

返回以指数表示法（e 表示法）表示的数值的字符串形式。

toPrecision
numObj.toPrecision(precision)

toPrecision() 方法以指定的精度返回该数值对象的字符串表示。
实际上，toPrecision() 会根据要处理的数值决定到底是调用 toFixed() 还是调用 toExponential()

字符方法

charAt

charAt() 方法以单字符串的形式返回给定位置的那个字符。

charCodeAt

返回字符编码。

字符串操作方法

concat

拼接字符串，返回拼接得到的新字符串，原字符串不变。

slice
str.slice(beginSlice[, endSlice])

slice() 方法提取字符串中的一部分，并返回这个新的字符串。
若传入负值，则把所有的负值都与字符串的长度相加。

substr
str.substr(start[, length])

substr() 方法返回字符串中从指定位置开始的指定数量的字符。
若传入负值，则负的第一个参数加上字符串的长度，负的第二个参数转换为 0。

substring
str.substring(indexA[, indexB])

substring() 返回字符串两个索引之间（或到字符串末尾）的子串。
若传入负值，则把所有的负值参数都转为为 0。

注意：改方法会将较小的数作为开始位置，较大的数作为结束位置。

字符串位置方法

indexOf
str.indexOf(searchValue[, fromIndex])
lastIndexOf
str.lastIndexOf(searchValue[, fromIndex])

这两个方法都是从一个字符串中搜索给定的字符串，然后返子字符串的位置（如果没有找到该字符串，则返回 -1 ）

单体内置对象

定义：由 ECMAScript 实现提供的、不依赖于宿主环境的对象，这些对象在 ECMAscript 程序执行之前就已经存在了。

即开发人员 不必显式地实例化内置对象，因为它们已经实例化了。

encodeURI()

主要用于整个 URI，不会对本身属于 URI 的特殊字符进行编码。

encodeURIComponent()

主要用于对 URI 中的某一段进行编码，会对它发现的任何非标准字符进行编码。

decodeURI()

只能对使用 encodeURI() 替换的字符进行解码。

decodeURIComponent()

只能对使用 encodeURIComponent() 替换的字符进行解码。
